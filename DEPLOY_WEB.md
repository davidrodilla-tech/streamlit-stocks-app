# Publicar la app en una URL (Streamlit Community Cloud)

## 1) Subir estos archivos a GitHub
- `app.py`
- `streamlit_stocks_app.py`
- `requirements.txt`

## 2) Crear despliegue en Streamlit Cloud
1. Ve a [https://share.streamlit.io](https://share.streamlit.io)
2. Inicia sesión con GitHub.
3. Clic en **New app**.
4. Selecciona tu repositorio y rama.
5. En **Main file path** usa: `app.py`
6. Clic en **Deploy**.

## 3) Resultado
- Streamlit te dará una URL pública para abrir la app desde cualquier navegador.

## Nota
- Si haces cambios en GitHub, Streamlit permite redeploy para actualizar la web.
